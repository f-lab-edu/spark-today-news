## 프로젝트 소개
#### 실시간 뉴스 데이터 처리
2023년 07월 04일 ~ 2023년 08월 21일


## 프로젝트 명세
### 기술 환경
**BE** : `Python3.9.14`

**CI/CD** : `Github` `VS code`

### 과정 소개
#### 데이터 크롤링
네이버 뉴스 데이터를 크롤링하여 데이터를 수집
#### 데이터 수집 및 적재
`Kafka API`를 이용해 수집한 Raw한 데이터를 NCP Object Storage에 저장

`Spark SQL` 에서 저장된 데이터를 받아 전처리를 진행
### 구현

`Django` 웹 개발 도구를 통해 데이터 시각화
